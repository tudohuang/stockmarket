# StockSim Pro - 模擬股市交易

這是一個基於 Node.js 開發的即時模擬股市交易系統，支援美股、台股、日股、虛擬貨幣及期貨數據。

## 🚀 快速開始

### 1. 安裝環境
請確保你的電腦已安裝 [Node.js](https://nodejs.org/) (建議 v16 或以上版本)。

### 2. 下載並安裝依賴
如果你是從 GitHub 下載此專案，請在終端機 (Terminal/PowerShell) 執行：
```bash
npm install
```

### 3. 啟動伺服器
執行以下指令來啟動遊戲：
```bash
npm start
```
啟動後，請在瀏覽器打開：`http://localhost:3000`

---

## 🛠️ 開發與管理

### 自動化腳本
專案內附帶了兩種類型的啟動腳本：
- **Windows**: 執行 `start-game.bat` 即可自動檢查環境並啟動。
- **Linux/Mac**: 執行 `./setup.sh` 進行安裝與啟動。

## ✨ 主要功能
- **全台唯一換算**：自動透過 Yahoo Finance 獲取即時 USD/TWD 匯率，買賣美股自動換算。
- **多樣化標的**：美股 (AAPL, TSLA)、台股 (2330.TW)、虛擬貨幣 (BTC) 等。
- **技術分析**：內建 MA20 均線趨勢判定與 RSI 強弱指標。
- **槓桿交易**：支援 1x - 10x 槓桿及保證金機制。
- **排行榜系統**：記錄每次重玩時的投資回報率。
